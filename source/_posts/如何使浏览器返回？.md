# 两种返回的方式

* history.back(-1)

  ```javascript
  history.back(-1)//直接返回当前页的上一页，数据全部消息，是个新页面
  ```

* history.go(-1)

  ```javascript
  history.go(-1)//也是返回当前页的上一页，不过表单里的数据全部还在
  ```



# 返回、前进页面的方法下面总结

```javascript
window.location.reload() //刷新
window.history.go(1) //前进
window.history.go(-1) //后退
window.history.forward() //前进
window.history.back() //后退+刷新
```
